# COMPACT STOCKER 画面仕様

## 1. 画面一覧

- ログイン
- ホーム
- サイドメニュー
- 出荷開始
- 出荷プラン一覧
- 出荷作業詳細
- 出荷実績検索
- 出荷実績一覧
- 在庫一覧
- 在庫履歴詳細

## 2. 画面仕様

### 2.1 ログイン

- 目的
  - 利用者を識別し、所属倉庫を確定する入口とする
- 入力
  - ユーザー ID
  - パスワード
- 現在の実装
  - 非空入力でログイン成功
- 将来方針
  - 認証結果から所属倉庫コードを取得する

### 2.2 ホーム

- 目的
  - ログイン後の起点画面とする
- 表示内容
  - 当日の状況カード
  - 棚卸日
- 将来方針
  - 所属倉庫ごとの件数を表示する

### 2.3 サイドメニュー

- 目的
  - 主機能への遷移
- 遷移先
  - ホーム
  - 出荷作業
  - 出荷実績
  - 在庫一覧

### 2.4 出荷開始

- 目的
  - 出荷作業の条件を指定する
- 入力
  - 出荷日
  - 出荷先コード
- 遷移
  - 出荷プラン一覧

### 2.5 出荷プラン一覧

- 目的
  - 対象出荷プランを選択する
- 表示内容
  - 出荷先名
  - 品目数
  - 出荷 ID 入力欄
  - QR 読み取り導線
- 遷移
  - 出荷作業詳細

### 2.6 出荷作業詳細

- 目的
  - 出荷対象商品の確認と実績入力
- 表示内容
  - ロケーションごとの商品一覧
  - 商品名
  - 商品コード
  - 予定数
  - 実績数
  - 差分
- 操作
  - 検索
  - QR 読み取り
  - 実績数量入力
- 完了条件
  - 全品目の実績入力完了

### 2.7 出荷実績検索

- 目的
  - 条件指定して実績を絞り込む
- 入力
  - 日付
  - ユーザーコード
- 遷移
  - 出荷実績一覧

### 2.8 出荷実績一覧

- 目的
  - 検索結果の確認
- 表示内容
  - 商品名
  - 数量
  - ユーザー
  - 日付
- 操作
  - 削除
  - CSV 出力

### 2.9 在庫一覧

- 目的
  - 現在在庫の一覧確認
- 表示内容
  - 商品名
  - 商品コード
  - ロケーション
  - 数量
- 操作
  - タップで詳細へ
  - 長押しでプレビュー表示

### 2.10 在庫履歴詳細

- 目的
  - 商品ごとの履歴を確認する
- 表示内容
  - 商品情報
  - 入出庫履歴
  - 実施日時
  - 実施者

## 3. 共通仕様

- 基本言語は日本語
- ナビゲーションは画面タイトルを明確に表示する
- サイドメニューから主要機能へ移動できる
- 倉庫確定後は、倉庫単位のデータを表示対象とする
