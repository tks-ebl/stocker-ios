# COMPACT STOCKER 管理者向け接続設定画面仕様

## 1. 文書の目的

この文書は、無料公開版の Azure 固定接続と、管理者による社内サーバー IP アドレス直指定を切り替えるための接続設定 UI を定義するものです。

実装時に、導線、表示文言、入力バリデーション、エラー表示がぶれないようにすることを目的とします。

## 2. 画面の位置付け

- 一般利用者向けの主要導線には置かない
- アプリ内に画面として持たせる
- サイドメニュー直下には置かない
- 設定画面の下層、または管理者向け項目として控えめに配置する
- 追加パスワードは設けない

補足:

- 追加パスワードがないため、画面導線は目立たせすぎないことを必須とする

## 3. 画面名

- 画面名
  - `接続設定`
- セクション名
  - `現在の接続先`
  - `社内サーバー設定`
  - `接続確認`

## 4. 表示項目

### 4.1 現在の接続先

- 項目名
  - `接続モード`
  - `Azure デモ環境`
  - `社内サーバー`
- 表示例
  - `接続モード: Azure デモ環境`
  - `接続モード: 社内サーバー`
- 接続先表示
  - Azure 利用時は固定 URL を表示専用で表示する
  - 社内サーバー利用時は `https://IPアドレス` を表示する

### 4.2 社内サーバー設定

- 入力項目
  - `社内サーバー IP アドレス`
- プレースホルダ
  - `例: 192.168.10.20`
- 補助文言
  - `社内サーバーを利用する場合のみ入力してください`
  - `接続先は https://IPアドレス で保存されます`
  - `固定 IP または予約 IP のサーバーを指定してください`
  - `証明書の SAN に指定 IP が含まれている必要があります`

### 4.3 ボタン

- `Azure デモ環境を使用`
- `社内サーバーを使用`
- `接続確認`
- `保存`
- `キャンセル`

## 5. 文言案

### 5.1 画面説明文

- `接続先を切り替えます。通常は Azure デモ環境を使用します。`
- `社内サーバーを利用する場合は、管理者が固定 IP の接続先を設定してください。`

### 5.2 保存前確認文言

- `この設定は管理者向けです。接続先を変更すると再ログインが必要です。`
- `社内サーバーを利用する場合は、同一ネットワーク上で接続できることを確認してください。`

### 5.3 Azure モード説明文

- `無料公開版の標準接続先です`
- `接続先 URL は変更できません`
- `デモ用の利用制限があります`

### 5.4 社内サーバーモード説明文

- `契約環境または社内サーバーに接続します`
- `接続先は https://IPアドレス で指定します`
- `固定 IP が必須です`

## 6. 入力バリデーション

- 入力必須
  - 社内サーバー利用時は IP アドレス入力を必須とする
- 形式チェック
  - IPv4 形式のみ初期対応とする
  - `https://` はアプリ側で自動付与し、手入力不要とする
- 保存不可条件
  - 空文字
  - 不正な IPv4 形式
  - ループバックアドレス
  - `0.0.0.0`
- 警告条件
  - プライベート IP 以外を入力した場合は警告を出す

## 7. 接続確認仕様

- `接続確認` 実行時に API の疎通確認を行う
- 確認先は `GET /health` を基本とする
- 接続確認成功時は保存可能とする
- 接続確認失敗時は保存不可または警告付き保存不可とする

### 7.1 成功メッセージ

- `接続を確認しました`
- `社内サーバーに接続できます`

### 7.2 失敗メッセージ

- `サーバーに接続できませんでした`
- `同じネットワークに接続しているか確認してください`
- `サーバーが起動しているか確認してください`
- `証明書設定を確認してください`
- `入力した IP アドレスを確認してください`

## 8. 保存時動作

- 保存時に接続モードを更新する
- 保存後は既存トークンを破棄し、再ログインを要求する
- 保存成功後は現在の接続モードを画面上に反映する

### 8.1 保存成功メッセージ

- `接続設定を保存しました。再ログインしてください`

### 8.2 保存失敗メッセージ

- `接続設定を保存できませんでした`
- `入力内容を確認してください`

## 9. Azure デモ制限の表示

- Azure モード選択時は以下を画面上に表示する
  - `Azure デモ環境の制限`
  - `1倉庫まで`
  - `2ユーザーまで`
  - `在庫100件まで`
  - `履歴参照は直近30日まで`
  - `CSV出力は利用できません`

## 10. 今後の拡張余地

- IPv6 対応
- ホスト名入力対応
- MDM 配布時の接続先自動設定
