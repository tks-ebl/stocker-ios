# COMPACT STOCKER 管理者向け接続設定画面仕様

## 1. 文書の目的

この文書は、開発用 / 公開用 / カスタマイズ用の接続先方針のうち、カスタマイズ用で利用する接続設定 UI を定義するものです。

実装時に、導線、表示文言、入力バリデーション、エラー表示がぶれないようにすることを目的とします。

## 2. 画面の位置付け

- 一般利用者向けの主要導線には置かない
- アプリ内に画面として持たせる
- 公開アプリ版では通常導線を持たせない
- カスタマイズ版でのみ有効化する
- 追加パスワードは設けない

補足:

- 追加パスワードがないため、公開アプリ版ではソース修正なしに表示できない構成を必須とする
- 開発時は必要に応じてこの画面を有効化し、ローカル URL や検証用 URL の切替確認に使う

## 3. 画面名

- 画面名
  - `接続設定`
- セクション名
  - `現在の接続先`
  - `社内サーバー設定`
  - `接続確認`

## 4. 表示項目

### 4.1 現在の接続先

- 項目名
  - `接続モード`
  - `開発用`
  - `公開用`
  - `カスタマイズ用`
- 表示例
  - `接続モード: 開発用`
  - `接続モード: 公開用`
  - `接続モード: カスタマイズ用`
- 接続先表示
  - 開発用はローカル URL を表示する
  - 公開用は Azure 固定 URL を表示専用で表示する
  - カスタマイズ用は入力済み URL を表示する

### 4.2 社内サーバー設定

- 入力項目
  - `接続先 URL`
- プレースホルダ
  - `例: https://192.168.10.20`
- 補助文言
  - `カスタマイズ用を利用する場合のみ入力してください`
  - `開発用と公開用の URL は固定値として扱います`
  - `社内サーバーを利用する場合は https://IPアドレス を推奨します`
  - `証明書の SAN に指定 IP が含まれている必要があります`

### 4.3 ボタン

- `開発用を使用`
- `公開用を使用`
- `カスタマイズ用を使用`
- `接続確認`
- `保存`
- `キャンセル`

## 5. 文言案

### 5.1 画面説明文

- `接続先を切り替えます。公開版では通常は公開用を使用します。`
- `カスタマイズ用を利用する場合は、管理者が接続先 URL を設定してください。`

### 5.2 保存前確認文言

- `この設定は管理者向けです。接続先を変更すると再ログインが必要です。`
- `社内サーバーを利用する場合は、同一ネットワーク上で接続できることを確認してください。`

### 5.3 公開用モード説明文

- `無料公開版の標準接続先です`
- `接続先 URL は変更できません`
- `デモ用の利用制限があります`

### 5.4 カスタマイズ用モード説明文

- `契約環境または社内サーバーに接続します`
- `接続先 URL は管理者が設定します`
- `社内サーバーを使う場合は固定 IP を推奨します`

## 6. 入力バリデーション

- 入力必須
  - カスタマイズ用利用時は URL 入力を必須とする
- 形式チェック
  - 初期対応は `https://` から始まる URL とする
- 保存不可条件
  - 空文字
  - `http://` URL
  - 不正な URL 形式

## 7. 接続確認仕様

- `接続確認` 実行時に API の疎通確認を行う
- 確認先は `GET /health` を基本とする
- 接続確認成功時は保存可能とする
- 接続確認失敗時は保存不可または警告付き保存不可とする

### 7.1 成功メッセージ

- `接続を確認しました`
- `接続先に接続できます`

### 7.2 失敗メッセージ

- `サーバーに接続できませんでした`
- `同じネットワークに接続しているか確認してください`
- `サーバーが起動しているか確認してください`
- `証明書設定を確認してください`
- `入力した URL を確認してください`

## 8. 保存時動作

- 保存時に接続モードを更新する
- 保存後は既存トークンを破棄し、再ログインを要求する
- 保存成功後は現在の接続モードを画面上に反映する

### 8.1 保存成功メッセージ

- `接続設定を保存しました。再ログインしてください`

### 8.2 保存失敗メッセージ

- `接続設定を保存できませんでした`
- `入力内容を確認してください`

## 9. Azure デモ制限の表示

- 公開用モード選択時は以下を画面上に表示する
  - `Azure デモ環境の制限`
  - `1倉庫まで`
  - `2ユーザーまで`
  - `在庫100件まで`
  - `履歴参照は直近30日まで`
